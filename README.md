### Build a simple quizzing app similar to menti meter

## Features

-Admin should be allowed to add quesions (MCQ, single answer)
-Admin should be allowed to move to the next quesions
-Admin should be allowed to show the leaderboard to everyone
-Users should be allowed to answer the qustions
-Users just need to poll the server for the next quesion , no need for it to be realtime
